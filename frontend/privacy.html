🔒 Política de Privacidade - SaaS Delivery

✓ Privacidade é importante para nós - Esta política explica como protegemos seus dados

1. Introdução

"SaaS Delivery" (doravante "Plataforma") é um serviço de software-as-a-service que respeita e protege a privacidade de seus usuários.

Esta Política de Privacidade explica:

  • ✓ Que dados coletamos
  • ✓ Como usamos esses dados
  • ✓ Quem tem acesso aos dados
  • ✓ Seus direitos quanto aos dados
  • ✓ Como protegemos seus dados

4. Dados Que NÃO Coletamos

  • ✅ Não coleta: Histórico de navegação fora da Plataforma
  • ✅ Não coleta: Dados de geolocalização em tempo real
  • ✅ Não coleta: Dados de câmera ou microfone
  • ✅ Não coleta: Histórico de redes sociais
  • ✅ Não coleta: Arquivos pessoais de seu computador
  • ✅ Não coleta: Senhas de suas outras contas

5. Compartilhamento de Dados

Seus dados SÃO compartilhados com:

  • 🤝 Seus clientes (quando fazem pedidos conosco)
  • 🤝 Provedores de pagamento (para processar pagamentos)
  • 🤝 Provedores de email (para enviar notificações)
  • 🤝 Equipe de suporte (quando você solicita ajuda)
  • 🤝 Autoridades legais (quando obrigado por lei)

Seus dados NÃO SÃO:

  • ❌ Vendidos a terceiros
  • ❌ Compartilhados com concorrentes
  • ❌ Usados para marketing sem consentimento
  • ❌ Transferidos para outros países (por padrão)
  • ❌ Combinados com dados de outros usuários

6. Segurança de Dados

Como protegemos seus dados:

  • 🔐 Criptografia SSL/TLS - Todos dados em trânsito são encriptados
  • 🔐 Banco de dados seguro - PostgreSQL com acesso restrito
  • 🔐 Senhas hasheadas - Nem nós vemos suas senhas
  • 🔐 Rate limiting - Previne ataques de força bruta
  • 🔐 Firewall - Bloqueia acessos não autorizados
  • 🔐 Backup automático - Seus dados não se perdem
  • 🔐 Monitoramento 24/7 - Detectamos atividades suspeitas
  • 🔐 HTTPS apenas - Sem conexões não-encriptadas
⚠️ Importante: Embora fazemos o máximo para proteger, nenhum sistema é 100% seguro. Se descobrir uma vulnerabilidade, reporte para: security@seusite.com

7. Retenção de Dados

8. Seus Direitos

Você tem o direito de:

  • 📋 Acessar - Solicitar uma cópia de todos seus dados
  • 📋 Retificar - Corrigir dados incorretos
  • 📋 Deletar - Solicitar exclusão de seus dados ("direito ao esquecimento")
  • 📋 Portabilidade - Exportar seus dados em formato legível
  • 📋 Reclamar - Entrar em contato com autoridade de proteção de dados
  • 📋 Revogar consentimento - Retirar autorização de coleta

Para exercer qualquer um desses direitos, envie email para: privacy@seusite.com com o assunto "Solicitação de Dados"
